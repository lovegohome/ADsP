1장. 데이터 이해   
	데이터 정보, 데이터 베이스(정의, 특징, 활용)   
      
2장. 데이터의 가치와 미래   
	빅데이터(이해, 가치, 영향) → 활용되는 비즈니스 모델(위기요인, 통제방안), 미래의 빅데이터   
   
3장. 가치창조를 위한 데이터 사이언스와 전략 인사이트   
	빅데이터가 사회와 기업에 미치는 영향 → 전략적 인사이트, 빅데이터 발전방향
